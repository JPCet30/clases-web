Lista de Componentes Comunes

  • Encabezado (Header): Sección superior con título o logo.
  • Barra de Navegación (Nav): Menú con enlaces para navegar.
  • Contenido Principal (Main): Área con el contenido principal.
  • Barra Lateral (Sidebar): Contenido secundario a un lado.
  • Pie de Página (Footer): Sección inferior con información adicional.
  • Botones: Elementos interactivos para acciones.
  • Formulario: Campos para entrada de datos.
  • Tarjetas (Cards): Contenedores para información agrupada.
  • Galería de Imágenes: Colección de imágenes organizadas.
  • Modal: Ventana emergente para contenido adicional.
