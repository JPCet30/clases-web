🎨 Guía Práctica: ¡Dale Estilo a tu Página Web con CSS!

👋 ¡Hola! ¡Bienvenido a tu primera aventura con CSS!

Piensa en esto:

  • HTML es como el esqueleto 💀 de una página web. Pone los textos, los títulos y las imágenes.
  • CSS es como la ropa y el maquillaje 👕💄. Le da color, tamaño y estilo.

Hoy, vamos a vestir nuestro esqueleto HTML para que se vea genial. ¡Usaremos Visual Studio Code para crear una pequeña tarjeta de presentación!

📂 Paso 1: Prepara tu Espacio de Trabajo

  1. Abre Visual Studio Code.
  2. Ve a Archivo > Abrir Carpeta y crea una nueva carpeta llamada Mi_Pagina_Web.
  3. Dentro de esa carpeta, crea dos archivos nuevos:
    • index.html (aquí vivirá nuestro esqueleto).
    • styles.css (aquí guardaremos toda la ropa y estilos).

뼈 Paso 2: Construye el Esqueleto (HTML)

Copia y pega este código en tu archivo index.html. Este es el contenido de nuestra página.

<!DOCTYPE html>
<html lang="es">
<head>
    <title>Mi Tarjeta de Presentación</title>
</head>
<body>
    <h1>Mi Nombre</h1>
    <p>Me encanta la programación y el diseño.</p>

    <div class="tarjeta">
        <h2>Sobre mí</h2>
        <p>Estoy aprendiendo a crear páginas web increíbles.</p>
    </div>
</body>
</html>

*Si abres este archivo en tu navegador, verás texto sin estilo. ¡Ahora vamos a darle vida!*

🔗 Paso 3: Conecta la Ropa con el Esqueleto (Link CSS)

Para que el HTML sepa dónde buscar sus estilos, debemos conectarlos.

En tu archivo index.html, agrega esta línea dentro del <head>:

<link rel="stylesheet" href="styles.css">

El <head> de tu index.html ahora se verá así:

<head>
    <title>Mi Tarjeta de Presentación</title>
    <link rel="stylesheet" href="styles.css">
</head>

¡Excelente! Ahora están conectados.

🎨 Paso 4: ¡A Pintar! Tu Primera Regla de CSS

Vamos a tu archivo styles.css. Aquí es donde ocurre la magia.

Una regla de CSS tiene 3 partes:

  1. Selector: ¿A qué le quieres dar estilo? (Ej: h1, p, body).
  2. Propiedad: ¿Qué quieres cambiar? (Ej: color, font-size).
  3. Valor: ¿Qué aspecto le quieres dar? (Ej: blue, 24px).

Cambiemos el color del título <h1> a un tono azul. Escribe esto en styles.css:

h1 {
    color: #4A90E2; /* Un bonito color azul */
}

¡Guarda ambos archivos! Ahora, abre o actualiza el archivo index.html en tu navegador. ¡Tu título principal ahora es azul!

🖌️ Paso 5: Cambiando el Fondo y el Texto de Toda la Página

Podemos cambiar el estilo de toda la página seleccionando la etiqueta <body>.

Agrega esto a tu archivo styles.css:

body {
    background-color: #F4F4F9; /* Un fondo gris muy claro */
    font-family: sans-serif;   /* Una fuente fácil de leer */
    color: #333333;           /* Color de texto principal (gris oscuro) */
}

¡Refresca tu navegador! Toda la página tiene un nuevo fondo y una letra más moderna.

🏷️ Paso 6: Creando una Tarjeta con "Clases"

En nuestro HTML, hay un <div> con algo especial: class="tarjeta". Una "clase" es como una etiqueta de nombre que le ponemos a un elemento para darle un estilo único.

Para darle estilo a una clase, usamos un punto . antes del nombre.

Agrega esto a styles.css para diseñar nuestra tarjeta:

.tarjeta {
    background-color: white;      /* Fondo blanco para la tarjeta */
    border: 1px solid #DDDDDD;    /* Un borde delgado y gris */
    border-radius: 10px;          /* ¡Esquinas redondeadas! */
    padding: 20px;                /* Espacio por dentro de la tarjeta */
    margin: 30px;                 /* Espacio por fuera de la tarjeta */
    box-shadow: 0 4px 8px rgba(0,0,0,0.1); /* Una sombra para que parezca que flota */
}

¡Guarda y refresca! Ahora deberías ver una hermosa tarjeta con esquinas redondeadas y una sombra sutil. ¡Se ve muy profesional!

🎉 ¡Felicidades! 🎉
